## Project Goals
This project is a modern, responsive, single-page website built for a local IT and networking supplier, Fuerza Digital. The goal is to showcase the company's product categories, provide company information, and allow users to contact the business while adhering to modern UI/UX and accessibility standards.

## Technologies Used
* **HTML5:** Semantic structuring (`<header>`, `<main>`, `<section>`, `<footer>`).
* **CSS3 & Bootstrap 5:** Utility-first framework used for the grid layout, responsive design, and mobile-first approach. Custom CSS used to ensure WCAG 4.5:1 color contrast.
* **JavaScript:** Used for mobile navigation toggling, tabbed content interactivity, and custom client-side form validation.

## How to Run Locally
1. Download or clone the repository to your local machine.
2. Ensure all files (`index.html`, `style.css`, the images, and `README.md`) are in the same folder.
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox), or use the VS Code "Live Server" extension for hot-reloading.
